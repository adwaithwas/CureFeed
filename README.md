# CureFeed

A minimal, intentional YouTube feed.

## Features
- Latest videos from selected channels
- No algorithm
- No infinite scroll
- Remembers what you watched

## Setup
1. Clone repo
2. Open channels.json
3. Add your channel IDs
4. `pip install -r requirements.txt`
5. `python app.py`

CureFeed is an inbox, not a feed.
